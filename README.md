
# Cliff Walking Problem Game Using SARSA Algorithm

## 🎮 Overview
This project implements the **Cliff Walking problem** from Reinforcement Learning using the **SARSA (State–Action–Reward–State–Action)** algorithm.  
The Cliff Walking environment is a classic gridworld scenario where an agent must learn to navigate safely from start to goal while avoiding the cliff.  

The goal of this project is to demonstrate how SARSA learns optimal policies through exploration and exploitation in a dynamic environment.

---

## 🧠 Key Concepts
- **Reinforcement Learning (RL):** Training an agent to maximize cumulative rewards.
- **SARSA Algorithm:** An on-policy temporal difference learning method.
- **Cliff Walking Environment:** A gridworld where falling off the cliff results in large negative rewards.

---

## ⚙️ Features
- Implementation of the Cliff Walking environment.
- SARSA algorithm for training the agent.
- Visualization of agent’s path and learning progress.
- Configurable parameters (episodes, learning rate, discount factor, epsilon).
